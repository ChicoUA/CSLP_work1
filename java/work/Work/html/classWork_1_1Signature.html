◆ signatureMaker1()

void Work.Signature.signatureMaker1 ( int  len)
inline

Function that creates a signature, given a String. First, it splits the string into shingles. Second, it uses each shingle, one each iteration, and creates a key, by getting the value of each letter and multiplying by the position of that letter in the shingle and adds to the key. Third, in that same iteration, it uses the key, and the values of the hash function and creates the code (code = (a * key + b) % p). Finally, the smallest code is the one saved inside the signature. This process is repeated for each hash function, the more functions, the bigger is the signature.

Parameters
lensize of the shingles
