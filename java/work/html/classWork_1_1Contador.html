◆ adicionar()

void Work.Contador.adicionar ( )
inline

Function used to increase the counter. If special is true, then the percentage to count will be reduced logarithmically for each iteration. It starts at 1(100%), then 1/2(50%), 1/3(~33%) ... And the number to add to the counter will increase, first is 1, then 2, 3, 4 ... Else it will count only with a percentage of 50% and increase the counter by 2.
